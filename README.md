# Binaural Soundwave Generator and Session Timer

A web app that generates binaural beats for focus, relaxation, meditation, or sleepâ€”with an optional session timer (work/break cycles) or a simple elapsed-time counter.

**Use headphones** for the binaural effect to work.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview   # serve the production build locally
```

## Tech

- React 19, TypeScript, Vite
- Web Audio API for stereo binaural playback and pink noise

## Deploy

The app is a static SPA. Build with `npm run build` and serve the `dist` folder. Works on [Vercel](https://vercel.com), Netlify, or any static host.
